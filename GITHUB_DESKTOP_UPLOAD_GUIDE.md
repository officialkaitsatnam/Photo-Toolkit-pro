# GitHub Desktop Upload Guide

## Smart Photo Toolkit Pro v41.7 Figma UI + Build-1A

### Steps
1. Is ZIP ko extract karein.
2. GitHub Desktop open karein.
3. `File > Add Local Repository` par click karein.
4. Extracted folder select karein.
5. Agar repository nahi bani hai to `Create a Repository` choose karein.
6. Commit message likhein: `v41.7 Figma UI Build-1A Aadhaar Printable`
7. `Commit to main` karein.
8. `Publish repository` ya `Push origin` karein.

### Important
- `index.html`, `main.css`, `main.js`, `script.js`, images, manifest aur service worker saath me upload karein.
- CDN internet se load hota hai: PDF.js, jsPDF, pdf-lib.
- Browser cache issue aaye to hard refresh karein ya version query clear karein.
