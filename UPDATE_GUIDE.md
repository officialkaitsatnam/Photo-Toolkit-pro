# v42.8 Upload Guide

1. Extract ZIP.
2. Upload all files to GitHub repository root.
3. Make sure `index.html`, `main.js`, `main.css`, `sw.js` are at root.
4. Wait for GitHub Pages deployment.
5. Hard refresh site or clear browser cache.

Test checklist:
- Document Studio opens and PDF editor works.
- Passport Photo opens.
- PDF Studio opens.
- Profile dropdown opens.
- Sign in / Signup / Logout work.
- Footer links open pages.
